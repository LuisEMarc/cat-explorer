# 🐱 Cat Explorer

Aplicación web desarrollada para explorar razas de gatos mediante consumo de APIs públicas, con una interfaz moderna inspirada en One UI (Samsung).

---

## 🌐 Demo

👉 https://luisemarc.github.io/cat-explorer/

---

## 🧠 Objetivo

El objetivo del proyecto fue practicar el consumo de APIs REST, manipulación dinámica del DOM y diseño de interfaces modernas orientadas a dispositivos móviles.

La aplicación permite explorar distintas razas de gatos, visualizar información relevante y experimentar una experiencia visual inspirada en aplicaciones nativas de Android.

---

## ✨ Características

* 🐱 Consulta dinámica de razas mediante The Cat API
* 🖼 Visualización de imágenes asociadas a cada raza
* 🌎 Información de origen de cada raza
* 🌙 Modo oscuro / claro
* 💾 Persistencia de preferencias mediante LocalStorage
* 📱 Diseño responsive para móviles, tablets y escritorio
* 🎨 Interfaz inspirada en One UI
* ⚡ Renderizado dinámico mediante JavaScript Vanilla
